Shims directory: /.sprite/bin

Shim behavior
- Shims minimally initialize the language manager only if needed.
- Shims prepend the active toolchain bin directory to PATH (only if missing) and exec the real binary.
- If your environment keeps that bin directory earlier than /.sprite/bin on PATH, subsequent commands resolve to the real binary and bypass the sprite shim.

Node.js
- Manager: nvm at /.sprite/languages/node/nvm
- Shims: node, npm, npx, corepack
- Manage versions: nvm install <version>; nvm use <version>; nvm alias default <version>
- Default installed: 22.20.0
